.gitignore

.gitignore은 git이 디렉토리에 있는 내용을 무시하도록 설정하는 파일이다.
실수로 무시하려던 파일을 이미 add 했거나 push 했다면, 이를 git이 해당 파일들을 무시하게 설정하는 것은 약간 골치 아파진다. 일단 해당 파일들을 다음과 같이 git staging에서 제거해야 한다.
"git rm --cached [파일이름]"
git status를 하면 해당 파일들이 add 되지 않는 것을 알 수 있다. 이제 .gitignore 파일을 제대로 설정하고 .gitignore 파일만 add하고 commit 한다. 사용하는 repos가 깃허브라면 바로 확인해보도록 하자. test는 해서 확인했는데, 글로 쓰려니까 뭐라고 써야 할지 잘 모르겠다.
